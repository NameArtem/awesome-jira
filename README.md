## Keyboard Shortcuts

Some useful available keyboard shortcuts:

### Global Shortcuts

|  Action                    |  Shortcut     |
|---                         |---            |
|  Open Notification         |      'g' then 'n'        |
|  Go to Dashboard           |      'g' then 'd'        |
|  Create Content            |      'c       |
|  Open Recently Viewed      |      'g' then 'r'       |
|  Quick Search              |      '/'   |

### Issue Actions

|  Action                    |  Shortcut     |
|---                         |---            |
|  Edit Issue                |      'e'        |
|  Assign Issue              |      'a'        |
|  Assign to me              |      'i'        |
|  Comment on issue          |      'm'        |
|  Share issue               |      's'        |
|  Edit issue labels         |      'l'        |
